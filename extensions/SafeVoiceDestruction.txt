Safe voice destruction
======================

Status
------
This is no longer a separate extension API. ForgeAudio exposes both unchecked
and checked voice destruction in the public voice API.

Public API
----------
    void forge_voice_destroy(ForgeVoice *voice);
    ForgeResult forge_voice_try_destroy(ForgeVoice *voice);

Behavior
--------
`forge_voice_destroy` destroys the voice and asserts if destruction is invalid.

`forge_voice_try_destroy` returns a `ForgeResult` so callers can handle invalid
destruction without relying on assertions. It is useful when a voice may still
be referenced by the graph, active processing, or other runtime state.
