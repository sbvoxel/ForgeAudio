Filter wet/dry mix
==================

Status
------
This is no longer a separate extension API. ForgeAudio includes wet/dry filter
mixing directly in `ForgeFilterParameters`.

Public API
----------
`ForgeFilterParameters` contains:

    ForgeFilterType type;
    float frequency;
    float one_over_q;
    float wet_dry_mix;

The same structure is used by:

    forge_voice_set_filter_parameters
    forge_voice_get_filter_parameters
    forge_voice_set_output_filter_parameters
    forge_voice_get_output_filter_parameters

Behavior
--------
`wet_dry_mix` is in the range `[0, 1]`.

`1.0f` applies the filtered signal fully. `0.0f` leaves the original signal
unchanged. Intermediate values blend the filtered and unfiltered signal.

The default value is `FORGE_AUDIO_DEFAULT_FILTER_WET_DRY_MIX`.
